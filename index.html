What does this expect from my Excel file?
Any sheet that has these columns (raw ERP names or cleaned-up names both work — matching is case-insensitive):
  • SSP (or Lot #)
  • Item#
  • StnTyp (Stone Type)
  • Shp (Shape)
  • Col (Color)
  • Units (carat weight)
  • Lab + Report# (or one combined "Lab Report Number" column)
  • L, W, D (Length / Width / Depth)
  • IT (optional) — SS = Single Stone, PP = Parcel Packets
  • Remarks (optional) — used for the MM line on PP rows
The MM line follows the IT column: SS rows show the L - W X D dimensions, while PP rows show whatever is in the Remarks column instead. If there's no IT column, every row falls back to dimensions as before. No need to delete other columns first — extra columns are ignored. Every row becomes one label, 40 to a page.
